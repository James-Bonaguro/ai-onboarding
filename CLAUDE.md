# ai-onboarding — Project Context

## What this repo is
A learning system that teaches non-technical professionals to use AI as a reasoning partner, not a search tool. Doctrine → Setup → Examples → Agents.

## Audience
Analytical professionals in finance, private credit, insurance, consulting. Smart, busy, skeptical of AI hype. Many have never customized an AI tool. Some are on enterprise Copilot with limited settings access.

## Voice
Direct. Field manual, not course. No hooks, no influencer language, no "unlock," no "game-changer." Short sentences. Anti-corporate. The reader should feel like a sharp peer wrote this in one sitting.

## Constraints
- Tool-agnostic in user-facing docs (Claude / ChatGPT / Copilot). Only get tool-specific when walking through actual UI steps.
- Never assume the reader knows what GitHub, a repo, a markdown file, or a CLI is.
- Never add new files unless explicitly instructed.
- Never restructure folders.
- Never touch /examples or doctrine/06 unless told.

## Quality bar
A private credit AVP with zero AI background should be able to land on the README, follow the sequence, and finish Rule 5 in under 45 minutes feeling like they learned something real.
